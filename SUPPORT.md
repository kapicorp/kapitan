# Getting Support

## Community chat

The best place for quick questions and informal discussion is the [`#kapitan`](https://kubernetes.slack.com/archives/C981W2HD3) channel on Kubernetes Slack. [Get an invite](https://slack.k8s.io/) if you are not already on the workspace.

## Documentation

- [Getting Started](https://kapitan.dev/getting_started/)
- [Core Concepts](https://kapitan.dev/pages/core_concepts/)
- [FAQ](https://kapitan.dev/FAQ/)
- [Kapitan Reference repository](https://github.com/kapicorp/kapitan-reference) — working examples

## GitHub

- **Issues** — [Bug reports and feature requests](https://github.com/kapicorp/kapitan/issues)
- **Discussions** — [Q&A and general conversation](https://github.com/kapicorp/kapitan/discussions)
- **Pull Requests** — [Contribute code or documentation](https://github.com/kapicorp/kapitan/pulls)

## Security

For security-related issues, see [SECURITY.md](SECURITY.md) instead of opening a public issue.

## Sponsorship

If your organization benefits from Kapitan, consider [sponsoring the project](https://github.com/sponsors/kapicorp) to help fund ongoing maintenance and development.
